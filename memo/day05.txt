배열(Array) : 저장공간의 나열

변수 : []		하나의 타입으로된 하나의 값만 저장할 수 있는 저장공간

배열 : [][][][][]...	하나의 타입으로된 하나의 값만 저장할 수 있는 저장공간의 나열
			(하나의 타입으로 된 여러 값을 저장할 수 있음.)

배열의 선언
	1. 자료형[] 배열명 = {값1,값2,..};
	int[] ar = {1,2,3,4,5}; //어떤 값을 넣을지 알 때 사용

	2. 자료형[] 배열명 = new 자료형[칸수];
	int[] ar = new int[5]; //어떤 값을 넣을지 모를나, 몇 칸 만들지는 알 때 사용 
				//값을 초기화를 안하면 기본값이 들어감

	3. 자료형[] 배열명 = null;
	int[] ar = null;
	//어떤 값을 넣을지도 모르고, 몇 칸 만들지도 모를 때 사용

new : 뒤에 나오는 저장공간을 새로 만들어주고 그 저장공간의 시작주소를 가져옴.

null : 주소의 디폴트값(어떤 주소가 들어갈지 모를 때 사용)

index
	배열을 생성할 때 각각의 저장공간에서 자동으로 붙여지는 번호
	0부터 순차적으로 증가하면서 부여됨.
	배열의 이름이 1개이기 때문에 이름만으로는 배열의 여러 저장공간에 접근할 수 없음. 
	그렇기 때문에 index를 이용하여 접근함.
	배열명[index]로 사용하고 이는 하나의 저장공간이므로 변수처럼 사용함.

배열의 특징
	배열은 한 번 만들어지면 크기가 변할 수 없음.

length
	배열을 생성할 때 자동으로 생성되는 저장공간의 수(배열의 길이)
	배열명.length로 사용함
