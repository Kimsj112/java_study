분개처리 - 삼항 연산자,if,switch....

* 조건문
	- if 문
	if(조건식){
		실행할 문장
	}

	- if ~ else 문
	if(조건식){
		실행할 문장
	}else{
		실행할 문장
	}

	- if ~ else if ~ else 문

	if(조건식) {
		실행할 문장
	} else if(조건식) {
		실행할 문장
	} else {
		실행할 문장
	}

	- if : ()안에 조건식이 true면 {블록}안에 코드가 실행됨

	- if ~ else : 위의 조건식이 모두 거짓이고 else if의 조건식 참이면 실행

	- else : 위의 조건식이 모두 거짓이면 실행


	- switch 문
	switch(변수명) {
		case 값1:
			실행할 문장;
			break;
		case 값2:
			break;
		.....
		default :
			실행할 문장;
			break;
	}

	// 변수명의 값이 참으로 값이 참이면 실행되는데 break;를 안해주면
	참이어서 끝이 아니라 밑에 코드들도 차례대로 실행됨.

-삼항 연산자, if, switch 비교
	1. 삼항 연산자 : 조건식이 1개 있을 경우 사용을 고려함.
	2. if : 조건식에 비교연산자를 사용하거나, 
	여러 조건을 논리 연산자로 연결할 때 사용을 고려함.
	3. switch : 하나의 변수에 저장된 값을 동등(==)비교할 때 사용을 고려함.
